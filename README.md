# Reprap-

Holder for the files relating to the upgrade of a prusa i2 to a 300x200x200 printable area.
included are

## printed upgrade parts 

   1.1. Z axis height increase 
   
   1.2. wobble decrease 
   
   1.3. bed extinsion 

## Firmware changes 

   2.1. fw update 
  
   2.2. mod fw for larger area 

## Electrical changes 

   3.1. upgrade to solar
  
   3.2. upgrade to heated bed
  
   3.3. redo wiring 
  
   3.4. standard wiring clolrs
